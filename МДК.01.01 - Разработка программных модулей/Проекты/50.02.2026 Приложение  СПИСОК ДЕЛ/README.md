1. Описание проекта
Небольшое desktop‑приложение с графическим интерфейсом на Python и PyQt6.
Приложение позволяет пользователю добавлять задачи в список дел, просматривать их, удалять выбранные элементы и очищать весь список.

Цель: показать базовые навыки разработки GUI‑приложения, работу с событиями и оформление проекта в Git.

2. Функциональность
Приложение умеет:

запускаться в отдельном окне;

принимать текст задачи через поле ввода;

добавлять задачу в список по кнопке «Добавить»;

удалять выбранную задачу по кнопке «Удалить выбранное»;

очищать весь список задач по кнопке «Очистить список» (с подтверждением);

отображать количество задач внизу окна.

3. Стек технологий
Язык программирования: Python 3.x

GUI‑библиотека: PyQt6

Система контроля версий: Git

4. Установка и запуск
Клонировать репозиторий:

bash
git clone <ссылка-на-репозиторий>
cd <папка-проекта>
(Рекомендуется) создать виртуальное окружение:

bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/macOS
source venv/bin/activate
Установить зависимости:

bash
pip install pyqt6
Запустить приложение:

bash
python main.py
5. User Flow
Краткий сценарий работы пользователя:

Пользователь запускает приложение.

Вводит текст задачи в поле ввода.

Нажимает кнопку «Добавить» — задача появляется в списке.

Для удаления задачи:

выбирает её в списке,

нажимает «Удалить выбранное».

Для полной очистки списка нажимает «Очистить список» и подтверждает действие.

Нажимает кнопку закрытия окна — выход из приложения.
